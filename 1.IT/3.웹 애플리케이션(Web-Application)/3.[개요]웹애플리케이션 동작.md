# 웹애플리케이션의 동작
## 동작 순서
1. 최초요청(Request)
  - 사용자가 원하는 웹페이지의 주소(https://...)를 입력하면 서버에 최초로 요청을 보내게 된다.
2. 최초응답(Response)
  - 최초요청을 받은 서버는 기본적인 웹 사이트의 구조를 담고있는 `HTML파일`을 사용자에게 응답(전달)해준다.
3. 추가요청
4. 추가응답
